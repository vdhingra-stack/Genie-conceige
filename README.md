# Genie-conceige

Node/TypeScript project for Genie Concierge tooling, including MCP server scripts and request payloads.

## Getting Started

1. Install dependencies:
   
pm install
2. Run scripts as needed from MCP Server/Scripts.

## Project Structure

- MCP Server/Scripts - server and utility scripts
- MCP Server/config - configuration files
- MCP Server/mcp-requests - request samples
